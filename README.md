# PlanPro

A React app for task and project management, built with a responsive UI.

## Features
- Task organization
- Clean, modern design

## Installation
1. Clone the repo:
   ```bash
   git clone https://github.com/harshues04/PlanPro.git
   cd PlanPro
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run locally:
   ```bash
   npm start
   ```
